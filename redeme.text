# GIT 的快速入门

##　什么是 GIT

- GIT 是一个源代码管理工具
- 是一个版本控制工具
- 可以不联网使用

## GIT的常用命令

 `git init` 

> **使用方法：**在需要版本控制的目录中打开命令行工具使用
>
> **作用**：在文件夹中创建一个`.git`的文件夹，可以记录文件夹中文件的变更信息，该目录默认不可见。

` .gitignore 文件夹`

> 该文件夹是用来存放不需要`git`跟踪的文件。
>
> 在里面写不需要的记录变更的文件名。

`git status`

用于查看本地仓库的状态，第一次查看是一些没有被跟踪的文件。

`git add`

添加本地暂存托管文件，可以将一个没有被跟踪的文件添加到跟踪列表

`git commit -m`

将本地的变化提交到本地的仓库，- m 后面写的是本次更改说明。

一般是完成一个小模块之后使用。

`git log`

查看提交日志。
